# MAA System Architecture

## Core modes

### 1. Individual mode
- Child trains at own pace
- Exercises can be repeated
- Results are saved per child

### 2. Class mode
- Teacher starts exercise for the whole group
- All children see the same task
- Teacher controls speed and lesson flow

## First MVP module
Visual numbers training

## Future modules
- Addition exercises
- Audio exercises
- Abacus recognition
- Teacher dashboard
- Student progress tracking
