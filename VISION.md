# Light Academy OS

Light Academy OS is a digital learning ecosystem for children, parents, teachers and administrators.

The first module is MAA Trainer – a mental arithmetic training tool for children in our academy.

## Purpose

The purpose of Light Academy OS is to help children develop:

- concentration
- memory
- thinking
- mental calculation skills
- financial literacy
- self-awareness
- responsibility
- creativity

## First product

The first product is MAA Trainer.

MAA Trainer helps children practice mental arithmetic through short daily exercises.

## Main users

- Student
- Parent
- Teacher
- Administrator

## Future modules

- MAA Trainer
- FMA – Financial Mindset Academy
- AAA – Self-Awareness Academy
- ASA – Future Success Academy
- School Edition

## Development principle

We start small.

First, we build one useful training tool for our own academy students.

Only after testing it with real children, we expand the platform to parents, teachers, schools and other programs.
