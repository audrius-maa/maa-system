# PRODUCT

## Light Academy OS

The first product of Light Academy OS is:

# MAA Trainer

The goal of MAA Trainer is to help children improve concentration, memory and mental calculation through short daily exercises.

---

# User Journey

Student opens the application.

↓

Logs in.

↓

Sees today's training.

↓

Completes the exercise.

↓

Receives feedback.

↓

Progress is saved.

↓

Returns tomorrow.

# Product Principles

Simple.

Fast.

Fun.

No distractions.

Children should be able to start training within 10 seconds.

Every training session should take about 10 minutes.

The application should motivate children to return every day.
